## 特徴

* 開始・終了のパラメータを設定することで、その間のフレームのパラメータを自動設定します。
* パラメータの変化量は、Easing関数やSin/Cos等、様々な設定が可能です。
* 新規作成に特化しています。ツクール上の戦闘アニメを読み込んで編集することは出来ません。  
  ※本ツールで作成した戦闘アニメは、json形式でセーブ&ロードが可能です。

## 動作環境

Windows専用です。PowerShellが動作する必要があります。

## インストール方法

[Releases](https://github.com/lpre-ys/tk2k-effect-conductor/releases) より、最新のインストーラ、またはZIPをダウンロードしてください。

### インストーラ

インストーラを実行し、指示に従ってインストールします。

### ZIP

解凍し、お好きなフォルダに配置してください。

フォルダ内の`EffectConductor.exe`を実行すると、起動します。

## 使い方

説明していない機能も多々あるので、色々と触ってみてください。

1. 戦闘アニメの素材データを読み込む  
  右上、`Drag & Drop or Click`と書かれた領域に、戦闘アニメの素材画像をドラッグアンドドロップする、または、クリックして選択することで、素材データを読み込みます。
2. 右枠、セルの設定を自由に弄って戦闘アニメを作る  
3. セルが足りない場合、下部`タイムライン`の`追加`ボタンを押して、表示セルを追加する。
4. 上部、名前を入力し、`COPY!!`ボタンを押す。
5. クリップボード上に戦闘アニメデータが入っているので、ツクール側の戦闘アニメデータに貼り付けする。

### COPY!!ボタンでエラーが出た場合

下記ドキュメントを参考に設定変更をしてみてください。

[COPY!!ボタンでエラーが出た場合](https://github.com/lpre-ys/tk2k-effect-conductor/blob/main/doc/security.md)
